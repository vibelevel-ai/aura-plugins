# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

In short: be respectful, assume good intent, and help make this a welcoming place to build.
Harassment, discrimination, and hostile behavior are not tolerated.

**Reporting.** Conduct concerns can be raised privately to the maintainers via the project
Discord or by opening a confidential issue. Reports are handled discreetly.

A note specific to this project: plugins handle people's local session data. **Treat user
data with the same respect you'd want for your own** — never exfiltrate raw content, never
weaken the redaction contract, and never add hidden telemetry. Violations of the trust
model are treated as conduct violations, not just bugs.
